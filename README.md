## AI Workflow Execution

FlowPilot allows users to create AI-powered workflows and execute them using local LLMs through Ollama.

Workflow
    ↓
Prompt
    ↓
User Input
    ↓
Ollama
    ↓
Result

Example:

Workflow:
Meeting Notes Processor

Input:
- Call customer tomorrow
- Send proposal
- Create invoice

Output:
1. Call customer tomorrow
2. Send proposal
3. Create invoice
